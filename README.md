🤖 Telegram Bot — Генератор картинок по тексту
Описание
Это Telegram-бот, который позволяет пользователю:

Сгенерировать уникальную картинку по текстовому запросу с использованием внешнего нейросервиса.
Получить промо-код на онлайн-курс для заработка.
Взаимодействовать с ботом через удобное меню и кнопки.

🚀 Быстрый старт
1. Клонируй репозиторий
git clone https://github.com/OlegLazukov/telebot.py.git
cd telebot.py
2. Установка зависимостей
pip install -r requirements.txt
3. Настрой переменные конфигурации
Создай файл config.py в корне проекта:

TOKEN = "ваш_telegram_бот_token"
url_api = "https://example.com/"  # базовый URL api внешнего сервиса
api_key = "ваш_api_key"
secret_key = "ваш_secret_key"
4. Запусти бота
python main.py

📝 Основные команды и функционал
Действие	Как использовать
Начать работу с ботом	/start
Сгенерировать картинку	Кнопка "Найти картинку!", затем введите ваш запрос
Получить промо-код и ссылку на курс	Кнопка "Хочу начать зарабатывать!"
Интерактивные кнопки с эмодзи	Ответы на inline-кнопки "Уверен" / "Передумал"

⚙️ Как работает генерация картинки:
1.Пользователь выбирает "Найти картинку!" и присылает текстовый запрос.
2.Бот отправляет этот запрос внешнему сервису (через класс Text2ImageAPI).
3.Сервис возвращает сгенерированное изображение в base64.
4.Бот декодирует и отправляет картинку пользователю.

🔑 Зависимости
pyTelegramBotAPI (telebot)
emoji
requests
Установи всё одной командой:

pip install pyTelegramBotAPI emoji requests
⚠️ Примечания
Не забывай сохранять свой токен и ключи API в секрете (не коммить их в публичный репозиторий!).
Если сервис генерации картинок недоступен, бот не сможет корректно обработать запрос.
Для отправки стикеров убедись, что файл с стикером (.tgs) лежит рядом с main.py или укажи правильный путь.
